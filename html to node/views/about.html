Introduction

This Node.js project is developed as part of an academic assignment to understand backend development using Express.js. It focuses on server-side routing and page rendering.

Why Node.js?

Node.js allows JavaScript to run on the server, making it possible to build fast and scalable web applications with a single language for both frontend and backend.

Tools & Technologies

  1. HTML – Page Structure
  2. CSS – Design & Layout
  3. JavaScript – Logic
  4. Node.js – Server Environment
  5. Express.js – Routing Framework

Future Scope

This project can be extended by adding databases, user authentication, REST APIs, and frontend frameworks like React or Vue.
